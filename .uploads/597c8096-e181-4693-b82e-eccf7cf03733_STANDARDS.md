## 2. 后端规范

### 2.1 PHP 代码规范

#### 2.1.1 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 类名 | PascalCase | `ProjectController` |

| 数据库字段 | 下划线分隔 | `user_id`, `created_at` |

- 缩进：**4 个空格**（不使用 Tab）
- 行长度：软限制 **120 字符**
- 文件末尾：**留一个空行**
- 命名空间后：**留一个空行**

### 2.2 数据库规范

#### 2.2.1 表设计规范

```sql
-- 1. 表名使用下划线分隔，复数形式
CREATE TABLE `project_tasks` (
    -- 2. 主键
    `id` BIGINT UNSIGNED NOT NULL AUTO_INCREMENT,

    -- 3. 公共字段
    `user_id` BIGINT UNSIGNED NOT NULL DEFAULT 0 COMMENT '创建者ID',
    `created_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `updated_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    `deleted_at` DATETIME NULL DEFAULT NULL COMMENT '删除时间',

    -- 4. 业务字段
    `project_id` BIGINT UNSIGNED NOT NULL DEFAULT 0 COMMENT '项目ID',
    `name` VARCHAR(255) NOT NULL DEFAULT '' COMMENT '任务名称',

    -- 5. 索引
    PRIMARY KEY (`id`),
    INDEX `idx_project_id` (`project_id`),
    INDEX `idx_user_id` (`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='任务表';
```

#### 2.2.2 字段命名规范

| 字段类型 | 命名规则 | 示例 |

| 外键 | `{table_singular}_id` | `project_id`, `task_id` |
| 创建者 | `user_id` 或 `{role}_user_id` | `user_id`, `assignee_user_id` |
| 时间戳 | `{action}_at` | `created_at`, `updated_at`, `deleted_at` |
| 软删除 | `deleted_at` | `deleted_at` |

```javascript
/**
 * 格式化日期
 * @param {Date|string|number} date - 日期
 * @param {string} format - 格式化模板
 * @returns {string} 格式化后的日期字符串
 *
 * @example
 * formatDate(new Date(), 'YYYY-MM-DD HH:mm:ss')
 * // => '2024-01-01 12:00:00'
 */
function formatDate(date, format = 'YYYY-MM-DD') {
  if (!date) return ''
  const d = new Date(date)

  const year = d.getFullYear()
  const month = String(d.getMonth() + 1).padStart(2, '0')
  const day = String(d.getDate()).padStart(2, '0')
  const hours = String(d.getHours()).padStart(2, '0')
  const minutes = String(d.getMinutes()).padStart(2, '0')
  const seconds = String(d.getSeconds()).padStart(2, '0')

  return format
    .replace('YYYY', year)
    .replace('MM', month)
    .replace('DD', day)
    .replace('HH', hours)
    .replace('mm', minutes)
    .replace('ss', seconds)
}
```

/**

export const projectApi = {
